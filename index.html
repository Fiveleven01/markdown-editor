📝 基本语法

  • 标题：# H1 ## H2 ... ###### H6
  • 粗体：**文字** 斜体：*文字* 粗斜体：***文字***
  • 删除线：~~文字~~
  • 无序列表：- * + 有序列表：1.
  • 任务列表：- [ ] 待办 - [x] 完成
  • 引用：> 内容 嵌套用 >>
  • 行内代码：`code` 代码块：```语言
  • 分割线：--- *** ___
  • 链接：[文字](url) 参考式：[文字][id]
  • 自动链接：<url>
  • 图片：![描述](url)
  • 转义：\* 内嵌 HTML：直接写标签

📊 扩展语法

  • 表格：| A | B | + 分隔行 |---|---|
  • 对齐：:---左 :---:中 ---:右
  • 脚注：文中[^1] + 文末[^1]: 内容
  • 定义列表：术语一行 + : 定义
  • 表情：:joy: :heart: :fire: :rocket: :tada: :+1: 等 100+

🔬 LaTeX 公式

  • 行内：$E=mc^2$ 或 \(公式\)
  • 块级：$$公式$$ 或 \[公式\]

📈 图表

  • Mermaid：```mermaid 流程图/时序图/饼图
  • PlantUML：```plantuml 或 @startuml...@enduml
  • Infographic：```infographic 数据信息图

🗾 其他

  • Ruby 注音：[文字]{注音} 或 [文字]^(注音)
  • 分隔符：・ ． 。 -

🎨 常用 CSS（贴到样式面板）

  • 字号：p { font-size: 16px; }
  • 行距：p { line-height: 2; }
  • 字体：* { font-family: "微软雅黑"; }
  • 主题色：a { color: #e74c3c; }
  • 标题居中：h1 { text-align: center; border-bottom: none; }
  • 标题装饰线：h1::after { content:''; display:block; width:50px; height:3px; background:#2ab1ee; margin:12px auto 0; }
  • 去代码块圆点：pre::before { display: none; }
  • 白色代码块：pre { background: #f5f5f5 !important; }
